## Hotkeys

|        **Hotkey**        | **Description**                     |
| :----------------------: | ----------------------------------- |
|   `Ctrl + K, Ctrl + 0`   | Collapse All                        |
|   `Ctrl + K, Ctrl + J`   | Expand All                          |
|    `Ctrl + Shift + L`    | Highlight All Occurrences of a Word |
| `Ctrl + ]` or `Ctrl + [` | Multiline Indent                    |
